Committee member please use the Proposal Admin Only button on the front page to login to the admin end of the module
The Admin page contains Four main items

Committee Members

The Committee Members button opens a list of all the unapproved proposal, this log page provides three main functions:
  1. The internal comments link, allow committee members to exchange comments among themselves which do not get sent to the PIs.
  2. The send comments link allows you to communicate with the PI send comments and view his response.(Please allow the committee to be the sole composer of all comments then send to PI)
  3. The approve link allows the committee to approve the proposal an email will be send out the PI.(Only Chair should finally approve)

Check Proposals

The Check and Route allows Mary to check the status of the operator, IACUC and Animal Transfer status and expirations, as a final step route, if the proposal is routed with of items not checked that implies it is not required.

Accounts

The Accounts button allows Eman to create accounts in one click.

Proposal Log

The Proposal Log button lists archived proposals.
